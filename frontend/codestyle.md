# Code specification - Team XP Contacts Frontend
This specification is based on Airbnb JavaScript Style Guide and Google HTML/CSS Style Guide.

## HTML specification

### Semantic tag
  Using semantic tags：header, main, section, footer.
  Add alt attribute to the picture.
  Form element association label.

### Code format
  Indent with 2 spaces
  Use double quotes for attributes
  Label lowercase

## CSS specification

### Naming specification
  Class names use lowercase letters and hyphens：.contact-card
  ID is named after hump：#mainContainer.

### Code organization
  Reset the style
  General style
  Layout style
  Component style
  Responsive style1

## JavaScript specification

### Variables and functions
  Variables and functions use small humps: fetchContacts.
  Constants are capitalized: API_BASE_URL.

### Code format
  Indent with 2 spaces
  Plus sign at the end of the statement
  Use single quotation marks
