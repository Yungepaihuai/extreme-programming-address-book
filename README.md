# Extreme Programming Address Book

## 项目简介
这是一个基于极限编程方法的地址簿管理系统，具有完整的CRUD功能、Excel导入导出、收藏联系人等特性。

## 项目结构
extreme-programming-address-book/
├── frontend/ # 前端代码
│ ├── css/ # 样式文件
│ ├── js/ # JavaScript文件
│ ├── index.html # 主页面
│ └── README.md # 前端说明
├── backend/ # 后端代码
│ ├── src/ # Java源代码
│ ├── pom.xml # Maven配置文件
│ └── README.md # 后端说明
└── README.md # 项目总说明

## 技术栈
- **前端**: HTML5, CSS3, JavaScript, SheetJS
- **后端**: Spring Boot, H2 Database, Apache POI
- **部署**: 华为云/阿里云服务器

## 快速开始
1. 克隆项目: `git clone https://github.com/Yungepaihuai/extreme-programming-address-book.git`
2. 启动后端: `cd backend && mvn spring-boot:run`
3. 启动前端: 在浏览器中打开`frontend/index.html`
